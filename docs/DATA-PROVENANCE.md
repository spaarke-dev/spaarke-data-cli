# Data Provenance

All document content in this demo dataset is **synthetically generated** using Claude AI. No real client data, no real court filings, no real contracts.

## Sources

| Source Type | Description | License |
|-------------|-------------|---------|
| Synthetic legal documents | AI-generated contracts, pleadings, memos, expert reports | Proprietary (Spaarke) |
| Synthetic email content | AI-generated legal correspondence in EML format | Proprietary (Spaarke) |
| Synthetic invoice data | AI-generated billing documents | Proprietary (Spaarke) |

## Entity Names
All entity names (Meridian Corp, Pinnacle Industries, Baker & Associates LLP, etc.) are fictional.
